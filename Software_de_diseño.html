PROPÓSITO DEL MÓDULO:

Diseña páginas web, animaciones multimedia, imágenes digitales e impresas, para generar productos de comunicación y publicidad en el ámbito laboral y comunitario, favoreciendo el desarrollo ético, creativo e intercultural del entorno.


Durante este módulo aprenderás los conceptos básicos para el desarrollo de software en un lenguaje de programación y el manejo adecuado de bases de datos.

Bloc de notas


Bloc de notas es un procesador de texto muy sencillo en el que se pueden crear archivos de cualquier tipo que contengan texto, por lo tanto tambien códigos. Inicialmente crearás págias web básicas con el Blog de notas, lo que te ayudará a comprender la estructura básica de una pagina web y los comandos más básicos, ademas de que aprenderás a estructurar un sitio web desde cero


Esta alicación ya viene por defecto en Windows. En caso de tener Mac puedes usar Mac Notepad



Sublime Text


Sublime text es un software creado especialmente para el diseño de páginas web. Una vez que ya sepas todos los conceptos básicos que aprendes en bloc de notas, se utilizará el software Sublime Text, lo que facilitará demasiado la creación de una página web y su mejor estructuración


Puedes obtener este software completamente gratis en la pagina de SublimeText.



Pagina Web Agenda 2030

Se realizó una página web con algún aspecto de la Agenda 2030 para el desarrollo sostenible, aplicando dicho tema en el contexto del COBACH siguiendo los criterios de la escala estimativa otorgada por el maestro





Página web de la capacitación

Como proyecto final se creó una página web generada con el lenguaje de etiquetas de HTML5 y con CSS3, dicha página sirve como promoción de la Capacitación, esta página debe ser creada siguiendo los criterios de la escala estimativa otorgada por el maestro

Vegas Pro


En este submódulo volverás a trabajar con Vegas Pro y explorarás más a fondo las herramientas del software para hacer una edición más profesional en las actividades de vide realizadas

Corel Draw


Corel Draw es un software de diseño digital, conocerás gran parte de las herramientas de ese programa para crear diversos anuncios publicitarios y propaganda de algún determinado tema o produco y podrás crear diseños oríginales para distintas cosas que te puedan servir en un futuro.


Pueden faciliarte la obtención de este software en el laboratorio de Informática.



Adobe Photoshop


En este submodulo, también aprenderás a manejar el software de Adobe Photoshop para la edición y creación de imágenes, además de tambien poder crear diseños oríginales con una mayo calidad con las herramientas que este programa ofrece.
