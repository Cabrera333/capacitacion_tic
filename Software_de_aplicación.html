Descripción General


Descargar contenido del módulo




PROPÓSITO DEL MÓDULO:

Prepara información en documentos electrónicos mediante programas de aplicación para resolver diferentes tipos de problemas presentes en su entorno, en donde se permita transmitir información de forma congruente, coherente y organizada.


Durante ese módulo se trabja en las aplicaciones del bloque básico de Microsoft office, tales como Word, Power Point y Excel. Aprenderás funciones que no probablemente conocías de cada software, lo que ayuda a tener una mayor eficacia a la hora de realizar trabajos.


Además de que durante este módulo puedes certificarte en dichas aplicaciones por medio de a plataforma de Testing Program.

Microsoft Office Word


En este submódulo trabajarás con el procesador de texto Microsoft Office Word, en el cual aprenderás a realizar distintos tipos de documentos y también aprenderas a utilizar distintas herramientas de este software con las cuales podrás realizar trabajos de manera más eficiente y puedas utilizarlos como todo un profesional.


Pueden faciliarte la obtención de este software en el laboratorio de Informática.



Testing Program


Para ayudar al aprendizaje en el procesador de texto, utilizarás la plataforma llamada Testing Program, en la que puedes realizar distíntas prácticas para poder conocer todas las herramientas de Microsoft Office Word.


Deberás pasar los exámenes aplicados por medio de esta plataforma para poder adquirir tu acreditación y certicifación en Microsoft Office Word Academic.



Revista

El proyecto final realizado en este submódulo es una revista creada por el estudiante con las herramientas del procesador de texto Microsoft Office Word, realizada completamente desde cero y siguiendo los criterios de la escala estimativa otorgada por el maestro.





Certificación

En este submódulo obtendrás tu acreditación y certificación en Microsoft Office Word

Microsoft Power Point


En este submódulo trabajarás creando presentaciones electrónicas con el software Power Point, en el cual realizarás distiantas actividades para conocer mejor las herramientas del programa y poder crer presentaciones electrónicas de calidad que serán muy útiles a la hora de presentar un tema a un grupo de personas.


Pueden faciliarte la obtención de este software en el laboratorio de Informática.



Testing Program


En este submódulo utilizarás la mísma plataforma que en el procesador de texto, Testing program, en la cual también realizarás distíntas prácticas de presentaciones electrónicas para ampliar tu conocimiento en el software Microsoft Power Point.


Deberás pasar los exámenes aplicados por medio de esta plataforma para poder adquirir tu acreditación y certicifación en Microsoft Power Point Academic.



Audacity


Audacity es un software de edición de audio el cual se utilizará en la creación y edición de efectos de audio para el contenido multimedia de las presentaciones electrónicas que se trabajan en este submódulo.


Puedes descargar este software de manera gratuita en la página de audacity.



Audio

Se realizó un proyecto hecho en Audacity, que consiste en un pequeño cuento narrado por el estudiante y edición, incluyendo efectos y música de ambientación, todo esto siguiendo las indicaciones de la escala estimativa otorgada por el maestro







Presentación Multimedia

El proyecto final de este submódulo es una presentación multimedia de algún tema de otra clase que tenga el alumno, aplicando todos los conocimientos adquieridos durante el submódulo, siguiendo las indicaciones de la escala estimativa otorgada por el maestro







Certificación

En este submódulo obtendrás tu acreditación y certificación en Microsoft Power Point

Microsoft Excel


En este submódulo trabajarás con con hojas de cálculo aplicado en el software Microsoft Excel, aprenderás a utilizar este programa y gran parte de las funciones que este posee, además de que aprenderás a organizar datos y crear bases de datos y formularios sencillos


Pueden faciliarte la obtención de este software en el laboratorio de Informática.



Testing Program


Para ayudar en el aprendizaje de hojas de cálculo aplicado, utilizarás la plataforma de Testing Progran, al igual que en los archivos de texto y las presentaciones electrónicas. Aquí también realizarás diversas actividades que te ayudaran a conocer más el entorno del softwre y las funciones del mismo.


Deberás pasar los exámenes aplicados por medio de esta plataforma para poder adquirir tu acreditación y certicifación en Microsoft Excel Academic.



Cuestionario

El proyecto final realizado en este submódulo es un cuestionario realizado en Microsoft Excel de el tema de alguna otra clase que tenga el alumno, utilizando los conocimientos adquiridos y siguiendo los criterios de la escala estimativa otorgada por el maestro.
