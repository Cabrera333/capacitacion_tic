Descripción General


Descargar contenido del módulo




PROPÓSITO DEL MÓDULO:

Integra Comunidades virtuales con la intención de generar la interacción del estudiantado, utilizando software social y educativo favoreciendo un ambiente de comunicación incluyente y ético dentro de su entorno personal, profesional y educativo.


Desarrolla habilidades para resolver problemas de operación en el equipo de cómputo, aplicando diferentes tipos de mantenimiento, empleando diferentes conexiones para las redes de cómputo, trabajando con responsabilidad y tolerancia a la frustración en los ámbitos organizacionales, educativos y de la vida cotidiana.


Durante este módulo aprenderás todo lo que conforma a una computadora y también a manejar comunidades virtuales.

Vegas Pro


Vegas Pro es un software de edición de video, el cuál utilizarás para hacer un video el cual posteriormente subirás a la plataforma de YouTube como parte de una comunidad virtual.


Pueden faciliarte la obtención de este software en el laboratorio de Informática.



Audacity


En este submódulo utilizarás el softuare audacity para la creación de un audio el cual servirá como Podcast para tu comunidad virtual


Puedes descargar este software de manera gratuita en la página de audacity.



Chamilo


Chamilo es una plataforma de e-learnin, en la cual aprenderás a crear un curso de algún tema en concreto como parte de una comunidad virtual educativa en la cual podrás programar diversas actividades como examenes.


Esta plataforma permite su uso de forma completamente gratuita.



Video

Se realizó un video para una comunidad virtual mostrando los componentes internos de ana camputadora en conjunto con el otro submódulo de este módulo siguiendo las indicaciones de la escala estimativa otorgada por el maestro.





Podcast

Se realizó un podcast con el software audacity del mísmo tema del video, esto para trabajar los dos submódulos del módulo en conjunto siguiendo las indicaciones de la escala estimativa otorgada por el maestro.







Curso Online

Para este submódulo también se realizó un curso en la plataforma de Chamilo sobre algúna materia elegida por el equipo como parte de una comunidad virtual educativa siguiendo las indicaciones de la escala estimativa otorgada por el maestro.

PC
