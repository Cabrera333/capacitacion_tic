PROPÓSITO DEL MÓDULO:

Plantea soluciones críticas y responsables mediante la metodología de desarrollo de software para demostrar eficiencia en el manejo de bases de datos y software de programación de alto nivel, que sean aplicables a necesidades de una empresa o institución para el tratamiento de la información.


Durante este módulo aprenderás los conceptos básicos para el desarrollo de software en un lenguaje de programación y el manejo adecuado de bases de datos.

Microsoft Excel


En este submódulo volverás a trabajar con Microsoft Excel para crear bases de datos y formularios más avanzados utilizando las herramientas vistas en el Modulo de Software de Aplicación y ademas haciendo uso de macros hechas con Visual Basic para Aplicaciones.


Pueden faciliarte la obtención de este software en el laboratorio de Informática.



Microsoft Access


Después de haber trabajado con Microsft Excel trabajarás con el software Microsoft Access. Aprenderás las distintas herramientas de este programa, lo que fcilitará en gran medida la creación de bases de datos y formularios, con más opciones de las que se tenía en excel.


Pueden faciliarte la obtención de este software en el laboratorio de Informática.



Sistema de información en Microsoft Excel

En la primer parte de este submódulo se utilizó el software de Microsoft excel para generar un Sistema de información haciendo uso de Macros, Formularios y programando con el elenguaje de programación VBA (Visual Basic para Aplicaciones), siguiendo los criterios de la escala estimativa otorgada por el maestro.





Sistema de información en Microsoft Access

En la seguna parte del submódulo se ulizó el programa Access para crear una base de datos con sus respectivos formularios haciendo uso de relaciones y de más herrmaientas que el software otorga par realizar un sstema de información más complejo siguiendo los criterios de la escala estimativa otorgada por el maestro.

Raptor


Raptor es un software en el que se pueden crear diagramas de flujo. Por medio de este software aprenderás los principios básicos de la progrmación y las estructuras básicas de un código.


Puedes obtener este software completamente gratis en internet.



Dev C++


Dev C++ es un software de desarrollo en el que se utiliza el lenguaje de programación C++. En este programa podrás crear diferentes códigos para crear distintas soluciones a problemas o desarrollar algunos sistemas. Por medio de este programa aprenderás estructuras más complicadas y aprenderás a combinarlas para poder llegar a una solución


Puedes obtener este software completamente gratis en internet.



Album de diagramas de flujo de Raptor

Como proyecto se recaban todas las actividades realizadas en el software Raptor y se imprimen para tener un album de diagramas al que posteriormente se le agregn los respectivos pseudocódigos de cada diagrma.
